Spielbeschreibung "Jumping Block"


Jumping Block ist ein Jump n' Run Platformer. Das Ziel des Spieles ist alle "Pick Ups" innerhalb des Levels aufzusammeln, wenn dies geschehen erschreint eine Win-Message.
Das Spiel beinhaltet ein Hauptmenü mit 4 Buttons. 3 davon sind die verschieden Schwierigkeitsgrade des Levels und der vierte ist der "Quit Game"Button. Die Steuerung des Cube Characters erfolgt über  'W,A,S,D' und Space Bar.
Desweitern gibt es in der "In-Game UI" einen Pick Up Counter sowie einen Button der  das Spiel restartet und einen Button ,der in das Hauptmenü  zurück führt. 



NOTE: Mir ist bewusst, das kein Sound bei dem Case eines Game Over und dem Case eines Wins abgespielt werden, diese Fehler waren nicht zu beseitgen!

Tipp Auf niedrigerer Auflösung Spielen, da sich das Hauptmenü verzieht und die in Game UI sehr klein ist.
